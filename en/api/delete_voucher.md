# Delete Voucher
**Request URL：**
- ` https://api.cloudshotspot.com/v2/voucher/delete `
  
**Request Method：**
- POST 

**Parameters:** 

|Parameter name|Require|Type|Note|
|:----    |:---|:----- |-----   |
|sitekey |  Yes  |    string   |    Sitekey   |
|secretkey |  Yes  |    string   |    Secretkey   |
|id |  Yes  |    string   |    Id   |



**Returned Example**

``` 
{
  "status": true,
  "msg": "success"
}

```
