1.8 删除账号

请求URL：

  • https://api.cloudshotspot.com/v2/voucher/delete

请求方式：

  • POST

参数：

参数名 必选 类型 说明
sitekey 是 string 无
secretkey 是 string 无
id 是 int 无
expire_date 是 string 无

返回示例

{
  "status": true,
  "msg": "success"
}
