
**请求URL：** 
- ` https://api.cloudshotspot.com/v2/account/update `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|sitekey |  是  |    string   |    无   |
|secretkey |  是  |    string   |    无   |
|usercode |  是  |    string   |    无   |
|truename |  是  |    string   |    无   |
|email |  是  |    string   |    无   |
|username |  是  |    string   |    无   |
|password |  是  |    int   |    无   |
|start_time |  是  |    string   |    无   |
|end_time |  是  |    string   |    无   |


**返回示例**

``` 
{
  "status": true,
  "msg": "success"
}

```
